Programação, nada mais é que o processo de escrita, teste e manutenção de um programa de computador, ou seja,há a criação de um conjunto de instruções que dizem ao computador, como realizar determinada tarefa. Os programas são escritos em uma linguagem de programação, embora seja possível escrevê-los diretamente em linguagem de máquina. Diferentes partes de um programa podem ser escritas em diferentes linguagens. De diversas formas ela é capaz de te transportar para outra realidade, é incrível.
